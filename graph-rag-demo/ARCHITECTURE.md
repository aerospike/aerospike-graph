## ğŸ“Š When Each Approach Wins

| Question Type | Best Approach | Why |
|---------------|---------------|-----|
| **Abstract concepts** (shortages, partnerships) | Vector/Hybrid | Graph can't extract entities from abstract terms |
| **Specific named entities** (CASGEVY, KISUNLA) | Graph/Hybrid | Graph finds correct cross-doc links |
| **Entity confusion risk** | Graph/Hybrid | Graph prevents hallucination via entity verification |
| **Cross-document synthesis** | Hybrid | Combines semantic similarity + entity relationships |
